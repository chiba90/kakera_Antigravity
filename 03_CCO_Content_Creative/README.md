# 03. CCO / Chief Content & Creative Officer (最高コンテンツ・クリエイティブ責任者)

ここはKakera JOURNALの「魂（コンテンツの質と世界観）」を守り、創り上げる総本山（制作・編集部）です。
SEOやマーケティングの数値（PV等）にとらわれず、「文章の深み（哲学）」と「視覚的な高級感（デザイン）」を極めることを目的とします。

## 役割とフロー
- 「どのような言葉選び、どのようなビジュアルで語るか」のブランド基準を定義します。
- `01_COO`（AIオペレーション）に対し、ここのルールをインプットとして与えて執筆させます。
- 出来上がった記事はまず `drafts/`（草稿）に置かれます。

## ディレクトリ構成
- `brand_identity/`: トーン＆マナー、アンチAI基準、ブランド哲学などのルールブック。
- `design_assets/`: 記事のOGP画像、動画用の静止画素材、CSSレイアウトなどの視覚的デザイン資産。
- `drafts/`: 執筆中の原稿、および `05_CQO` による監査が完了する前のすべての一次成果物。
