# CMO Strategies (マーケティング戦略室)

ここは、コンテンツをどのように世に広め、検索エンジン（SEO）やAIモデル（LLMO/AIO）に最適化するかという「方針」を策定・保管する場所です。

- 実作業を行うAI（COO・オペレーション部門）は、ここのガイドラインやキーワードリストを読み込んで原稿を作成・リライトします。

## 格納すべきファイル例
- `SEO_guidelines.md`: 狙うべきキーワード、H1/H2タグのルール、文字数などの規定。
- `LLMO_guidelines.md`: AIの検索エンジン（Perplexity等）が構造を理解しやすいマークアップのルール。
- `social_media_rules.md`: 各SNS（X, Instagram, LinkedIn等）への投稿フォーマットやトーン。
