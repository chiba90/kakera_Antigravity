# SNS戦略：前提共有と全体像の提示

## 1. トレンド・インサイト
現代のビジネス環境（ジョブ型、リモート普及など）において、阿吽の呼吸は通用せず、「前提の言語化」と「全体像の提示」がリーダーシップの土台となっています。今回のCEOのインサイトは、このマクロトレンドに対して「センスではなく訓練である」という極めて実戦的かつ泥臭いカウンターを提示するものです。

## 2. コミュニケーション戦略
- **目的**: 「点ではなく全体像から話す」というコミュニケーションの本質を、自身の反省（自己受容）と共に言語化し、読者に「これは訓練で改善できる」という前向きな気づきを与える。
- **ペルソナ**: マネジメント層、チームのコミュニケーションに課題を感じているビジネスパーソン、AI活用を模索する層。
- **トーン＆マナー**: Kakera×SNSハイブリッド型。AI特有の劇的な演出（「実は〜」「嘘のように〜」）を完全排除。代わりに「そりゃそうだ」「〜だもんな」といった千葉氏特有の独白・自己受容のリズムを取り入れ、説教臭くない等身大の気づきとして発信する。

## 3. アプローチ
1. **ファクトの提示**: 同じ情報でも前提が違えば受け取り方が違うという事実。
2. **自己へのベクトル**: いきなり「点」から話してしまう自分への反省（そりゃそうだ、の受容）。
3. **パラドックスの提示**: これはセンスではなく「訓練」でカバーできるという気づき。
4. **AIの活用**: 人間だけでなく、AIにも「前提・全体像」を整理させるという実践的なアクション。
