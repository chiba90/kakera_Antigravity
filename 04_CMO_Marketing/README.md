# 04. CMO / Chief Marketing Officer (最高マーケティング責任者)

ここは、出来上がったコンテンツを世に広め、検索エンジン（SEO）やAIモデル（LLMO/AIO）に最適化するための「流通・戦略部」です。
制作そのものは行わず（CCOの領域）、完成品をいかに市場へデリバリーするかに特化します。

## サブディレクトリ構成
- `strategies/`: SEOキーワードリスト、LLMO向けマークアップ指針など、「どうすれば読まれるか」の戦略ファイル。AI（COO）はここの戦略を読み込んでリライト等の作業を行います。
- `published_outputs/`: CCO（制作）が作り、CQO（品質管理）の監査をパスした、**最終的に世に公開された（公開が確定した）コンテンツ** のショーケースです。
