# 05. CQO / CRO Office (最高品質・最高リスク管理責任者)

ここは、AIが生成したコンテンツが世に出る前に、「ファクトチェック（ハルシネーションの排除）」と「リスク管理（ブランドセーフティ）」を行うための査読（監査）組織です。
出版業界における「校閲部」や「法務・コンプライアンス部」にあたります。

## 役割とフロー
1. **[生成]** AI（COO領域）が原稿を作成する。
2. **[査読]** 出来上がった原稿は、一度この組織（`review_queue/` など）に置かれ、品質チェック・ファクトチェックを受ける。
3. **[承認/公開]** リスクがないと判断されたものだけが、最終成果物として `03_CMO_Marketing/outputs/`（マーケティング部門）に引き渡され、世に公開される。

## デスクトップ構成
- `guidelines/`: リーガルチェック基準、NGワードリスト、ファクトチェックのSOP（手順書）を格納します。
- `review_queue/`: CMOにパスする直前の、査読待ちの草稿を置くための検疫部屋です。
